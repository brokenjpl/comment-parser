# comment-parser
The comment parser supports various languages based on FSM. It's written by scala and java.

### Supported Languages

- JAVA
- PY
- C
- CPP
- C_HEADER
- CPP_HEADER
- SCALA
- RUBY
- GO
- JS
- HTML
- BAT
- SH
- XML
- TEXT
- MD
- ETC

### Build environment

- Java 1.8
- Scala 2.12
- Sbt 1.0

### Compile & Test

- sbt compile
- sbt test
